# 📖 Read JSON in Rust

A simple Rust project that demonstrates how to read and parse JSON data using Rust. This project showcases file handling and JSON deserialization, making it a great starting point for working with structured data in Rust.

## 🚀 Features

* Read JSON data from a file
* Parse JSON into Rust data structures
* Handle errors gracefully
* Simple and beginner-friendly implementation

## 🛠️ Built With

* **Rust**
* **Serde**
* **Serde JSON**

## 📂 Project Structure

```text
read-json/
├── src/
│   └── main.rs
├── Cargo.toml

└── README.md
```

## ⚡ Getting Started

### Clone the Repository

```bash
git clone https://github.com/tushar434434/read-json.git
cd read-json
```

### Build the Project

```bash
cargo build
```

### Run the Project

```bash
cargo run
```
